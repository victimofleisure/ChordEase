*** ChordEase source distribution ***

REQUIREMENTS:

ChordEase is compatible with the following versions of Visual Studio:
MFC 6.0, VS2005, VS2008, VS2010, and VS2012.

INSTALLATION:

The source distribution of ChordEase doesn't have an installer.

UNINSTALLING:

To uninstall ChordEase, you need to do two things:

1.  Delete ChordEase's program files from the folder you copied them to.
2.  Delete ChordEase's registry entries, using regedit.  The key to
delete is:

"HKEY_CURRENT_USER\Software\Anal Software\ChordEase"

NOTES:

The binary version of this release is available from the ChordEase web
site, http://chordease.sourceforge.net/.
